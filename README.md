# 📓 Notebook

> **Write it down. Make it happen.**

Notebook is a simple and convenient note-taking app. Quickly and easily write down your thoughts, tasks, and ideas.

---

## ✨ Features

- **📝 Add Notes** — Create a new note with a title, content, and tag
- **🏷️ Tags** — Categorize each note into one of 4 categories:
  - 💼 **Work** — Professional tasks and responsibilities
  - 👤 **Personal** — Personal notes
  - 💡 **Ideas** — Ideas and plans
  - 🔔 **Reminders** — Reminders and alerts
- **🔍 Search** — Quickly search notes by title or content
- **🗂️ Filter** — Sort notes by tag
- **🗑️ Delete** — Delete unwanted notes with confirmation
- **💾 Storage** — All data is saved in `localStorage` (persists after page refresh)
- **📱 Responsive Design** — Adapts to mobile, tablet, and desktop screens

---

## 📖 How to Use

### Add a new note

1. Click the **"Add"** button in the top right
2. Enter a **title** and **content**
3. Select a tag: Work, Personal, Ideas, or Reminder
4. Click **"Save Note"**

### Delete a note

1. Click the 🗑️ icon on the note card
2. Click **"Delete"** in the confirmation dialog

### Search & Filter

- Type a keyword in the search box — results update automatically
- Select a tag from the dropdown to filter by category

---

## 📌 Note

All notes are stored **only in your browser**. Transferring to another device or backup is not currently available.
